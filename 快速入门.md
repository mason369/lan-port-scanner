# 快速入门指南

## 安装

### Windows 用户

1. 确保已安装 Python 3.7+
2. 双击运行 `启动.bat`
3. 程序会自动安装依赖并启动

### Linux/macOS 用户

1. 确保已安装 Python 3.7+
2. 在终端运行：
```bash
chmod +x 启动.sh
./启动.sh
```

### 手动安装

```bash
# 克隆仓库
git clone https://github.com/mason369/lan-port-scanner.git
cd lan-port-scanner

# 安装依赖
pip install -r requirements.txt

# 启动图形界面
python app.py
```

## 使用教程

### 1. 启动应用

运行 `python app.py` 启动图形界面。

### 2. 配置扫描参数

**网络配置：**
- 点击 🔄 按钮自动检测本地网络
- 或手动输入网络段（如 192.168.1）
- 设置主机范围（如 1-254）

**端口配置：**
- 常用端口：扫描 18 个最常用的端口
- Top 100：扫描最常见的 100 个端口
- 自定义：输入自定义端口列表（如 80,443,8080 或 1-1000）

**扫描参数：**
- 并发连接数：10-500（默认 100）
- 超时时间：0.5-10 秒（默认 1.0）
- 隐蔽模式：启用后使用随机化和延迟技术

### 3. 开始扫描

点击 "🚀 开始扫描" 按钮，程序将：
1. 验证输入参数
2. 显示扫描确认（大规模扫描时）
3. 开始扫描并实时显示结果
4. 更新统计信息

### 4. 查看结果

**表格视图：**
- IP 地址：目标主机 IP
- 端口：开放的端口号
- 状态：端口状态（open）
- 服务：识别的服务名称
- Banner：服务 banner 信息
- 时间：扫描时间

**详细信息：**
- 双击任意行查看完整的端口详情

**统计信息：**
- 活跃主机数量
- 开放端口总数
- 扫描耗时
- 扫描状态

### 5. 导出结果

**方式一：导出文件**
1. 点击 "💾 导出" 按钮
2. 选择保存位置和格式（JSON/CSV）
3. 确认保存

**方式二：复制到剪贴板**
1. 点击 "📋 复制" 按钮
2. 结果已复制，可粘贴到其他应用

### 6. 清空结果

点击 "🗑️ 清空" 按钮清空所有扫描结果。

## 常见问题

### Q: 扫描速度慢？
A:
- 增加并发连接数（100-200）
- 减少超时时间（0.5-1.0 秒）
- 禁用隐蔽模式
- 减少扫描的主机或端口数量

### Q: 扫描不到某些端口？
A:
- 检查防火墙设置
- 增加超时时间
- 确认目标主机在线
- 检查网络连接

### Q: 如何避免被检测？
A:
- 启用隐蔽模式
- 降低并发数（20-50）
- 增加超时时间
- 分批次扫描

### Q: 程序无法启动？
A:
- 确认 Python 版本 3.7+
- 运行 `pip install -r requirements.txt`
- 检查是否有其他程序占用端口
- 查看错误日志

## 命令行模式

如果需要在脚本中使用，可以使用命令行模式：

```bash
# 扫描本地网络的常用端口
python cli.py --auto --common

# 扫描指定网络
python cli.py -n 192.168.1 -r 1-100 --top100

# 自定义端口扫描
python cli.py -n 192.168.1 -r 1-50 -p 80,443,8080

# 导出结果
python cli.py --auto --common -o results.json
```

## Python 模块使用

```python
import asyncio
from scanner import StealthScanner, COMMON_PORTS

async def scan():
    scanner = StealthScanner()
    results = await scanner.scan_network(
        network_prefix="192.168.1",
        ports=COMMON_PORTS,
        host_range=(1, 254)
    )

    for ip, ports in results.items():
        print(f"主机 {ip}:")
        for result in ports:
            print(f"  端口 {result.port}: {result.service}")

asyncio.run(scan())
```
