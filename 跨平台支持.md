# 跨平台支持说明

## 支持的平台

本应用基于 Python 和 Tkinter 开发，支持以下平台：

- ✅ Windows 7/8/10/11
- ✅ Linux (Ubuntu, Debian, CentOS, Fedora, Arch 等)
- ✅ macOS 10.12+

## 平台特性

### Windows

**优势：**
- 原生 Vista 主题支持
- 完整的字体支持
- 一键启动脚本 (.bat)
- 无需额外配置

**安装：**
```bash
# 双击运行
运行.bat
```

**依赖：**
- Python 3.7+ (自带 tkinter)
- 无需额外安装图形库

### Linux

**优势：**
- 轻量级运行
- 完整的终端集成
- 脚本化支持

**安装：**
```bash
# Ubuntu/Debian
sudo apt install python3 python3-tk python3-pip
chmod +x 运行.sh
./运行.sh

# CentOS/RHEL
sudo yum install python3 python3-tkinter python3-pip
chmod +x 运行.sh
./运行.sh

# Arch Linux
sudo pacman -S python python-pip tk
chmod +x 运行.sh
./运行.sh
```

**依赖：**
- Python 3.7+
- python3-tk (tkinter 图形库)
- python3-pip

**注意事项：**
- 需要图形界面环境 (X11/Wayland)
- SSH 连接需要 X11 转发: `ssh -X user@host`
- 无图形环境可使用 CLI 模式

### macOS

**优势：**
- 原生 Aqua 主题
- 优雅的界面效果
- 完整的 Retina 支持

**安装：**
```bash
# 使用 Homebrew
brew install python3
chmod +x 运行.sh
./运行.sh
```

**依赖：**
- Python 3.7+ (通常自带 tkinter)
- Xcode Command Line Tools

## 平台特定说明

### Windows 特定

**文件路径：**
- 使用反斜杠: `C:\path\to\file`
- 支持 UNC 路径: `\\server\share`

**权限：**
- 某些操作可能需要管理员权限
- 右键 -> 以管理员身份运行

**防火墙：**
- Windows Defender 可能拦截扫描
- 添加应用到白名单

**编码：**
- 默认使用 UTF-8
- 支持中文路径和文件名

### Linux 特定

**文件路径：**
- 使用正斜杠: `/path/to/file`
- 区分大小写

**权限：**
- 扫描 1-1024 端口可能需要 root 权限
- 使用 `sudo` 或 capabilities

**图形环境：**
- 需要 X11 或 Wayland
- SSH: `ssh -X user@host`
- 无图形环境使用 CLI

**防火墙：**
- iptables/firewalld 可能拦截
- 配置防火墙规则

### macOS 特定

**文件路径：**
- 使用正斜杠: `/path/to/file`
- 区分大小写（默认不区分）

**权限：**
- 可能需要授予网络访问权限
- 系统偏好设置 -> 安全性与隐私

**Gatekeeper：**
- 首次运行可能被拦截
- 右键 -> 打开

**Retina 显示：**
- 自动支持高 DPI
- 界面清晰锐利

## 常见问题

### Q: Linux 上提示 "no display name and no $DISPLAY environment variable"？

**原因：** 没有图形环境或 X11 未配置

**解决方案：**
```bash
# 方案 1: 使用本地图形环境
startx

# 方案 2: SSH X11 转发
ssh -X user@host

# 方案 3: 使用 CLI 模式
python cli.py --auto --common
```

### Q: macOS 上提示 "Python is not installed as a framework"？

**原因：** Python 不是 framework 构建

**解决方案：**
```bash
# 使用 Homebrew 安装的 Python
brew install python3

# 或使用 pythonw
pythonw app.py
```

### Q: Windows 上中文显示乱码？

**原因：** 控制台编码问题

**解决方案：**
```bash
# 在脚本开头已设置 UTF-8
chcp 65001

# 或使用图形界面（无乱码）
python app.py
```

### Q: 不同平台界面大小不一致？

**原因：** DPI 缩放设置不同

**解决方案：**
- 应用会自动适应系统 DPI
- 可手动调整窗口大小
- 设置会保存（计划中）

## 开发说明

### 跨平台开发原则

1. **使用标准库**
   - 优先使用 Python 标准库
   - 避免平台特定的 API

2. **路径处理**
   - 使用 `os.path` 或 `pathlib`
   - 不要硬编码路径分隔符

3. **编码处理**
   - 统一使用 UTF-8
   - 明确指定编码

4. **GUI 设计**
   - 使用 ttk 而非 tk
   - 测试不同平台的主题

5. **测试**
   - 在所有目标平台测试
   - 使用虚拟机或 CI/CD
