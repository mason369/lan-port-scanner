# 启动说明

## 快速启动

### 方式 1: 简单启动（推荐）

```powershell
# 双击运行
run_simple.bat
```

这个脚本会：
1. 自动安装 PyQt6 和依赖
2. 启动应用

### 方式 2: 使用虚拟环境

```powershell
# 双击运行
start.bat
```

这个脚本会：
1. 创建虚拟环境
2. 安装依赖
3. 启动应用

### 方式 3: 手动安装并运行

```powershell
# 1. 安装依赖
pip install PyQt6 tabulate

# 2. 运行应用
python app_qt.py
```

## 如果遇到问题

### 问题 1: "系统找不到指定的路径"

**原因**: 虚拟环境路径问题

**解决方案**: 使用简单启动方式
```powershell
.\run_simple.bat
```

### 问题 2: "ModuleNotFoundError: No module named 'PyQt6'"

**原因**: PyQt6 未安装

**解决方案**: 手动安装
```powershell
pip install PyQt6 tabulate
python app_qt.py
```

### 问题 3: pip 安装很慢

**解决方案**: 使用国内镜像源
```powershell
pip install PyQt6 tabulate -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 问题 4: 权限不足

**解决方案**: 以管理员身份运行
- 右键点击 `run_simple.bat`
- 选择"以管理员身份运行"

## 依赖说明

本应用需要以下依赖：

- **PyQt6** (约 100MB)
  - 现代化的 GUI 框架
  - 跨平台支持
  - 专业的 UI 组件

- **tabulate** (约 50KB)
  - 用于 CLI 模式的表格格式化

## 系统要求

- Windows 7/8/10/11
- Python 3.7 或更高版本
- 约 150MB 磁盘空间（包含依赖）

## 启动时间

- 首次启动: 1-3 分钟（需要下载安装 PyQt6）
- 后续启动: 2-5 秒

## 验证安装

检查 Python 和依赖是否正确安装：

```powershell
# 检查 Python
python --version

# 检查 PyQt6
python -c "import PyQt6; print('PyQt6 installed')"

# 检查扫描器模块
python -c "from scanner import StealthScanner; print('Scanner OK')"
```

## 完整安装步骤

如果所有方式都失败，请按以下步骤操作：

```powershell
# 1. 确认 Python 已安装
python --version

# 2. 升级 pip
python -m pip install --upgrade pip

# 3. 安装依赖
pip install PyQt6 tabulate

# 4. 测试导入
python -c "import PyQt6; print('Success')"

# 5. 运行应用
python app_qt.py
```

## 卸载

如果需要卸载：

```powershell
# 卸载依赖
pip uninstall PyQt6 tabulate -y

# 删除虚拟环境（如果有）
rmdir /s venv

# 删除项目文件夹
cd ..
rmdir /s 端口扫描软件
```
